### 1.3 协议分类

java.net包下常用网络协议支持

#### TCP传输控制协议

- Transmission Control protocol
- **面向连接**的通信协议，3次握手（客户端和服务端三次交互）保证连接的可靠性
- 应用广泛，数据安全

#### UDP用户数据包协议

- User Datagram Protocol
- 无连接通信协议，简单快捷
